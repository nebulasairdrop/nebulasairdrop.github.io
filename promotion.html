星云空投推广计划

介绍

为了宣传和推广星云空投应用，赢得星云激励计划的最高奖金，我们制定了星云空投推广计划。与星云激励计划的奖励周期一致，我们在本网站上每周设置一个空投，空投的代币用于计算分红，分红资金来源于星云空投该周的在星云激励计划中的获得的奖金。同时如果获得月最佳应用的奖励，则综合该月的四个星期内总共空投币的数量，进行额外的分红。

规则

我们每个奖励周期拿出最多奖金的20%奖励在期间参与星云空投的用户。

假设空投的代币的缩写为 NPT（NASPromotionToken）

  1. 在一次空投中每个地址只能领取一次；
  2. 基础空投的数量是每个地址100 NPT；
  3. 填写有效的推荐人的地址可以获得额外 40 NPT；
  4. 每推荐一个朋友且成功领取空投后可以获得额外 40NPT；
  5. 一个周期结束后，当星云空投获得星云官方的奖励后（第二周周五24时），我们将在2天内自动把该周期内的分红分给所有参与空投的用户。

分红算法

在每个奖励周期内的分红的算法如下：

假设一个周期内，星云空投在星云激励计划中获得了N个星云币（NAS），空投代币总数量为M，基线为B=100000，则每个代币的价值为：

NPT = 0.2 * N / (B+ M) 个NAS

上面的公式有下面的规律：

  1. 每个周期内，随着空投代币总数的增加，用于奖励用户的奖金的比例也变大，当空投代币总数等于基线B时，比例为10%，随着总数无限增大，比例接近于20%，用户需要积极推广才能获得更大的奖励比例；
  2. 用户可能创建小号刷空投，但是刷空投降低了NPT的单位价值，而且考虑每笔交易需要付1-2元的手续费，因此当单次空投代币价值低于1元时，用户通过刷空投是不经济的行为，作弊行为不可能持续存在；
  3. 通过用户自发地刷空投和传播可能会有利于星云空投获得更高的奖励，但是低余额的账户刷空投对星云空投的帮助是很有限的。

金主奖

介绍

为了宣传和推广星云空投应用，赢得星云激励计划的最高奖金，我们制定了金主奖。与星云激励计划的奖励周期一致，我们在本网站上每周设置一个金主奖，金主奖的代币做为权重用于计算本周"最豪金主"，奖励资金来源于星云空投该周的在星云激励计划中获得的奖金。同时如果获得月最佳应用的奖励，则以月作为周期选出本月"最豪金主"。

规则

我们每个奖励周期拿出奖金的10%设立金主奖，奖励本周期内"最豪金主"。

假设空投的代币的缩写为 GMP（GoldMasterPrize）

  1. 在一次金主奖竞选周期内每个地址只能领取一次；
  2. 基础空投的数量是每个地址1 GMP；
  3. 填写有效的推荐人的地址可以获得额外 0.0001 GMP；
  4. 每推荐一个朋友且成功领取空投后可以获得额外 0.0001 GMP；
  5. 一个周期结束后，当星云空投获得星云官方的奖励后（第二周周五24时），我们将在2天内自动把该周期内的"最豪金主"奖励发给该周期的"最豪金主"。

确定"最豪金主"的算法和补充规则

在每个金主奖竞选周期内的算法如下：

假设一个周期内有完整的N天(每天星云链产生5760个区块)，星云官方公布的本周期第一个区块高度为(H + 1)。以金主A1为例，假设金主A1有 G 个 GMP，Balance(A1, H)表示的是A1账号在区块高度为H时的NAS余额，金主A1的本周期总积分M为：

M = G * (Balance(A1, H + 5760) + Balance(A1, H + 5760 * 2) + ... + Balance(A1, H + 5760 * N)) / N

本周期的"最豪金主"即为通过上面公式计算的M值最大的账号

对算法的说明和补充：

  1. 每个周期内，每个账号GMP的值基本都约等于1，通过邀请朋友对GMP的值仅仅产生较小的影响，而M的值更取决于竞选金主们在本周期内的账户平均余额；
  2. 用户可能创建小号刷GMP的值，但是这种行为很容易被发现，如果发现账号有通过自动化脚本实现"不诚实"的邀请的行为则取消该账号的竞选资格；
  3. "最豪金主"每周期只竞选一名，M值最高的是本周期的"最豪金主"，不同周期"最豪金主"可以重复获奖。
  4. 为了激励金主广泛参与,我们每个奖励周期再拿出奖金的10%奖励本周期内M值最大的第2-10名金主(比例基于M值)。
